Spotify Query


Seach Instructions:

Spotify Query allows you to search Spotify's database of music information. This query allows for three different types of searches based on the: Artist, Album, Track

The Artist search will return the amount of followers that the searched artist has on Spotify. To search, enter an artist/band name.

The Album search will return all of the albums that the searched artist/band has on Spotify. To search, enter and artist/band name.

The Tracks search will return the names of the tracks of the searched album on Spotify. To search, enter an album title.

* Spotify requries the use of a Access Token in order to use their API. The access token can be entered below the search query. The access token expires after one hour of being issued. In order to refresh the token click the Refresh Token Button and login in to Spotify to get a new access token.
